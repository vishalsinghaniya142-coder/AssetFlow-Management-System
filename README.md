# AssetFlow Management System

## Overview
AssetFlow Management System is an Odoo-based application developed for efficient asset management. It helps organizations track assets, monitor their status, manage configurations, and provide a centralized dashboard for quick insights.

---

## Features

### Dashboard
- Asset summary
- Active asset count
- Maintenance overview
- Status monitoring
- Quick statistics

### Settings
- Configure asset preferences
- Default asset values
- Dashboard settings
- User-specific configurations

### API
- REST API endpoints
- JSON responses
- Asset data integration
- External application support

### Controllers
- Backend routes
- Portal routes
- API routes
- Request handling

### Menu
- Dashboard Menu
- Settings Menu
- Easy Navigation

### Testing
- Unit testing
- Model validation
- Controller testing
- API testing

## Installation

1. Copy the module into the Odoo custom addons directory.

2. Restart the Odoo server.

3. Update Apps List.

4. Install **AssetFlow Management System**.
